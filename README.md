# NetCore6Template

## Installation

Clone the Repository
```
git clone https://github.com/PoppedBit/NetCore6Template.git
```

### Install and Build the React Apps

Login App
```
cd DotNetCore6Test/UI/login
npm install
npm run build
```

Client App
```
cd DotNetCore6Test/UI/client
npm install
npm run build
```
